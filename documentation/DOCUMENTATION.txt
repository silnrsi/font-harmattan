DOCUMENTATION
Harmattan
========================

This file describes the documentation files included with the Harmattan 
font. This information should be distributed along with the 
Harmattan font and any derivative works.

Primary documentation for the Harmattan font is on the Harmattan 
website (http://scripts.sil.org/Harmattan).

The Harmattan-features.pdf file illustrates the use of OpenType and 
Graphite features.

The Harmattan-features.odt file illustrates the use of OpenType and 
Graphite features (source file for the pdf).
